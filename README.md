## Usage

**handler-ses**
```
{
  "ses": {
    "mail_from": "sensu@example.com",
    "mail_to": "monitor@example.com",
    "use_ami_role": true,
    "access_key": "myaccesskey",
    "secret_key": "mysecretkey",
    "region": "us-east-1",
    "subscriptions": {
      "subscription_name": {
        "mail_to": "teamemail@example.com"
      }
    }
  }
}
```

**handler-sns**
```
{
  "sns": {
    "topic_arn": "arn:aws:sns:us-east-1:111111111111:topic",
    "use_ami_role": true,
    "access_key": "MY_KEY",
    "secret_key": "MY_secret"
  }
}
```
